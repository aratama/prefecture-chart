# 都道府県別総人口推移

RESAS API から人口データを取得してグラフに表示するウェブアプリです。

## 開発サーバーの起動

ローカル環境で開発サーバーを起動するには、以下のコマンドを実行してください。

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

それから、ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

# テスト

テストを実施するには、以下のコマンドを実行してください。

```bash
$ npm test
```
